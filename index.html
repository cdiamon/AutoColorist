AutoColorist

Это бесплатный проект с открытым исходным кодом для колористов.

View on GitHub Download .zip Download .tar.gz Download APK

Приложение под Android для колористов.

Спасибо за интерес к этому проекту. Приложение пока находится в состоянии Бета-тестирования. Чтобы установить себе программу, необходимо скачать APK-файл по ссылке сверху, залить его себе в смартфон, открыть его через стандартное приложение-проводник. После чего оно установится. Буду рад предложениям и замечаниям. Контакты внизу. Спасибо!

Каталог-галерея цветов и кодов по различным маркам и моделям автомобилей.

Это удобный каталог марок и моделей машин с реализованным умным поиском. Каждая ссылка отправляет на отдельную страницу со списком цветов и фотографиями данной модели авто. Каталог ссылается на онлайн ресурс, необходимо интернет-соединение.

Таблица переводов

Таблица переводов с поиском по таблице SQL по множеству популярных брендов: Spies Hecker, DuPont, Profiline, Mobihel, Normex, Brulex, Challenger, Duxone, EasiCoat, DeBeer, Sikkens, Kapci, Vika, Standox, Lesonal, NoMix, Reiz, PPG, QuickLine, Lechler, GreenLine.
